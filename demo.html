OrnaJS - tool that adds style for HTML elements without CSS and with no direct JavaScript in use.
It shows structure of OrnaJS. For example we make text red.

class=" color_red " - red color

class=" color_red_div " - red color to all child div

class=" color_red_div,a " - red color to all child div and a

class=" color_red_.classname " - red color to all child elements with class "classname"

class=" color_red_click " - red color on click

class=" color_red_div_click " - red color to all child div on click(Working on child div click)

class=" color_red_click_div " - red color to all child div on click (Working on parent element click)

class=" color_red_div_over " - red color to all child div on mouseover

class=" color_red_div_out " - red color to all child div on mouseout

class=" color_red_thisdiv_over " - red color to current child div on mouseover

Orna4Chrome it's Browser extension that automatically include orna.js script to your page. It's good for developer. You also don't need include it by your self, during development process. Just install it to Chrome or Opera, create directory and name it orna, than create html file with orna classes. When you run your file, Browser will understand that it OrnaJS page, because page in orna folder.
